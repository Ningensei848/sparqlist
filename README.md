# SPARQList

## Prerequisites

* [Node.js](https://nodejs.org/) v7.4.0
* [yarn](https://yarnpkg.com/) v0.19.1

## Install

    $ git clone git@github.com:enishitech/sparqlist.git
    $ cd sparqlist
    $ yarn install

## Run

    $ PORT=3000 yarn start

## Configuration

All configurations are done with environment variables.

### `PORT`

(default: `3000`)

Port to listen on.

### `REPOSITORY_PATH`

(default: `./repository`)

Path to repository.
